Quick Start

هذا الدليل يساعدك على الابتداء فوراً مع مشروع قبو الذكاء الاصطناعي المعزز دون الحاجة إلى تفاصيل تقنية معقدة.

  • متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
  • محرر نصوص (VS Code, Sublime Text, إلخ)
  • اختياري: Node.js للميزات المتقدمة
# انسخ المشروع إلى جهازك
git clone https://github.com/My-Knowledge-Garden/AI-Augmented-Vault.git

# انتقل إلى مجلد المشروع
cd AI-Augmented-Vault

# ثبّت المكتبات المطلوبة (اختياري)
npm install

  1. افتح الملف الرئيسي

    • انقر على index.html وافتحه في متصفحك
    • ستظهر واجهة تفاعلية لاستكشاف المحتوى

  2. تصفح الفئات الأربع

    • 🎨 الإنتاج الإبداعي: للموسيقى والفيديو والتصميم
    • 💭 الخطاب الجذري: للكتابة التحليلية والفلسفة
    • 🔧 الحلول التقنية: للبرمجيات والإصلاحات
    • 📚 إدارة المعرفة: لنظم التعلم

ابدأ بـ categories.md للحصول على قائمة شاملة منظمة

إذا كنت تريد إنشاء ملف manifest.json (قائمة شاملة بكل المحتوى):

# ثبّت المكتبة المطلوبة
npm install gray-matter

# شغّل السكريبت
node generate-manifest.js

# سيتم إنشاء manifest.json تلقائياً

لا تحاول قراءة كل شيء دفعة واحدة. ركز على موضوع واحد أولاً وسِر من هناك.

كل ملف يحتوي على روابط إلى ملفات ذات صلة. استخدمها للتعمق أكثر.

الفرغ ملاحظاتك الخاصة وأضفها إلى الملفات (إذا كنت تعمل نسختك الخاصة).

المحتوى يتطور باستمرار. تحقق من التحديثات الجديدة بشكل دوري.

إذا وجدت خطأ أو لديك اقتراح، شارك ملاحظاتك عبر GitHub Issues.
