Requirements

قبل البدء، تأكد من توفر المتطلبات التالية على جهازك:

  • Windows: Windows 10 أو أحدث
  • macOS: macOS 10.13 أو أحدث (Intel أو Apple Silicon)
  • Linux: أي توزيعة حديثة (Ubuntu 18.04+، Fedora 30+، إلخ)
  • المعالج (CPU): معالج حديث (Intel i5 / AMD Ryzen 5 أو أقوى)
  • الذاكرة (RAM): 4 GB على الأقل (8 GB موصى به)
  • مساحة التخزين: 500 MB على الأقل للمشروع الأساسي
  • الاتصال بالإنترنت: مطلوب للتثبيت والتنزيلات

يجب أن يكون لديك متصفح ويب حديث:

  • Google Chrome (إصدار 90+)
  • Mozilla Firefox (إصدار 88+)
  • Apple Safari (إصدار 14+)
  • Microsoft Edge (إصدار 90+)

أي من المحررات التالية لتحرير الملفات:

  • Visual Studio Code (موصى به) - مجاني وقوي
  • Sublime Text - خفيف الوزن وسريع
  • Atom - مرن وقابل للتخصيص
  • NotePad++ - بسيط وفعال
  • أي محرر نصوص آخر

مطلوب للميزات المتقدمة مثل إنشاء ملف Manifest:

  • النسخة الموصى بها: Node.js 14.0 أو أحدث
  • التنزيل من:
  • التحقق من التثبيت:
    node --version    # يجب أن يظهر v14.0.0 أو أحدث
npm --version     # يجب أن يظهر 6.0.0 أو أحدث

للحصول على آخر نسخة من المشروع:

  • النسخة الموصى بها: Git 2.30+
  • التنزيل من:
  • التحقق من التثبيت:
    git --version     # يجب أن يظهر git version 2.30+

{
  "gray-matter": "^4.0.3"  // لاستخراج بيانات Markdown

تُثبت تلقائياً عند تشغيل:

npm install

للميزات المتقدمة، قد تحتاج:

  • Python 3.8+ - للسكريبتات البرمجية
    • pip install للحزم المطلوبة
    • تثبيت من:
  • متصفح ويب: 100+ MB
  • Node.js: 100+ MB
  • المشروع نفسه: 50+ MB (بعد التنزيل)

إذا كنت تستخدم ميزات متقدمة:

  • خادم محلي: Python أو Node.js
  • خدمات الذكاء الاصطناعي الخارجية: اتصال API إلى خوادم خارجية
  • مخزن سحابي: GitHub، Google Drive، إلخ (اختياري)
  • القراءة: لكل الملفات في المشروع
  • الكتابة: لمجلد المشروع (لإنشاء manifest.json وملفات أخرى)
  • التنفيذ: للسكريبتات (generate-manifest.js, build_vault.py)
  • متطلب اختياري لتنزيل التحديثات من GitHub
  • متطلب اختياري للخدمات الخارجية والمكتبات

بعد تثبيت المتطلبات، تحقق من:

# تحقق من Node.js
node -v
npm -v

# تحقق من Git
git --version

# جرّب استنساخ المشروع
git clone https://github.com/My-Knowledge-Garden/AI-Augmented-Vault.git
cd AI-Augmented-Vault
npm install

# تحقق من أن كل شيء يعمل
node generate-manifest.js

إذا رأيت رسائل خطأ، راجع قسم الاستكشاف في troubleshooting.md.

  • مكتبات صوتية (اختياري): ffmpeg

  • لغات برمجة: Python, JavaScript, إلخ
