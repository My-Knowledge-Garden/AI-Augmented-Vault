Visual Arts

الفنون البصرية تشمل كل ما تراه بعينك - من التصوير إلى الفيديو إلى التصميم ثلاثي الأبعاد.

المكونات الأساسية:

  • 📸 التصوير: لقط لحظات جميلة
  • 🎬 الفيديو: سرد قصص متحركة
  • 🎨 التصميم: إنشاء بصريات جذابة
  • 🎭 الرسوم المتحركة: إحياء الحركة

الفتحة (Aperture):

يتحكم في:
├─ كمية الضوء الداخلة
├─ عمق المجال (Depth of Field)
└─ حدة التفاصيل

الرموز:
├─ f/1.8: فتحة واسعة (خلفية ضبابية)
├─ f/5.6: وسط
└─ f/16: فتحة ضيقة (كل شيء حاد)

سرعة الغالق (Shutter Speed):

يتحكم في:
├─ المدة الزمنية لدخول الضوء
├─ تجميد الحركة
└─ تأثير الحركة (Motion Blur)

الرموز:
├─ 1/1000: تجميد حركة سريعة
├─ 1/60: حركة عادية
└─ 1/2: تأثير حركة

حساسية الضوء (ISO):

يتحكم في:
├─ حساسية المستشعر للضوء
├─ دخول الضوء
└─ الضوضاء في الصورة

القيم:
├─ 100-400: عادي (قليل الضوضاء)
├─ 800-1600: مرتفع (ضوضاء معتدلة)
└─ 3200+: عالي جداً (ضوضاء كثيرة)

قاعدة الأثلاث:
├─ اقسم الصورة لتسع مربعات (3x3)
├─ ضع الموضوع الرئيسي على الخطوط
└─ يعطي توازن وتوتر بصري

المقياس:
├─ استخدم عناصر لإظهار حجم الموضوع
└─ مثل: شخص بجانب بناء

الألوان:
├─ الألوان المتكاملة (مسيرة مع بعضها)
├─ الألوان المتناقضة (عكسية)
└─ الألوان المتجاورة (بجانب بعضها)

الحركة والتوتر:
├─ أسطر رأسية: قوة واستقرار
├─ أسطر أفقية: سلام وهدوء
└─ أسطر قطرية: حركة وحيوية

Soft Lighting (إضاءة ناعمة):
├─ إضاءة منتشرة
├─ ظلال ناعمة
└─ مشهد هادئ وحنون

Hard Lighting (إضاءة قاسية):
├─ إضاءة مركزة
├─ ظلال حادة
└─ مشهد درامي وقوي

Backlighting (إضاءة خلفية):
├─ مصدر ضوء من الخلف
├─ هالة حول الموضوع
└─ تأثير درامي جميل

Three-Point Lighting (ثلاث نقاط):
├─ Key Light (الضوء الرئيسي)
├─ Fill Light (ضوء ملء الظلال)
└─ Back Light (ضوء خلفي)

Warm (دافئ): أصفر وأحمر
└─ يعطي شعور بالراحة والدفء

Cool (بارد): أزرق وأخضر
└─ يعطي شعور بالبرودة والهدوء

Neutral (محايد): رمادي وأبيض
└─ يعطي توازن بصري

Pre-Production (قبل التصوير):
├─ كتابة السيناريو
├─ التخطيط اللوحي (Storyboard)
├─ تجهيز الموقع
└─ إعداد المعدات

Production (التصوير):
├─ تصوير المشاهد
├─ تسجيل الصوت
├─ تسجيل المقابلات
└─ تصوير لقطات إضافية

Post-Production (بعد التصوير):
├─ تحرير الفيديو
├─ إضافة مؤثرات
├─ تحرير الصوت
└─ الإتقان النهائي

Cut (القطع):
├─ انتقال مباشر بين مشهدين
├─ الأكثر استخداماً
└─ واضح وسريع

Fade (التلاشي):
├─ تلاشي تدريجي من مشهد لآخر
├─ يعطي إحساس بالانتقال
└─ مناسب لنهايات الأقسام

Dissolve (الذوبان):
├─ مزج بين مشهدين
├─ أخف من الـ Fade
└─ انتقال سلس

Transition Effects:
├─ Wipes, Slides, Zoom
├─ مؤثرات بصرية
└─ استخدم بحذر

Resolution:
├─ 1080p (Full HD): عادي
├─ 2K: عالي الجودة
├─ 4K: احترافي جداً
└─ 8K: المستقبل

Frame Rate:
├─ 24fps: سينماتي
├─ 30fps: فيديو عادي
├─ 60fps: حركة سلسة
└─ 120fps+: حركة بطيئة

Aspect Ratio:
├─ 16:9: معيار الفيديو
├─ 9:16: هاتفي (Vertical)
├─ 1:1: مربع (وسائل تواصل)
└─ 2.39:1: سينماتي (عريض جداً)

1. Concept Art (الفن الاستكشافي)
   └─ رسومات تقريبية أولية

2. Blocking Out (البلوكات)
   └─ أشكال بسيطة للبنية الأساسية

3. Detailing (التفاصيل)
   └─ إضافة تفاصيل دقيقة

4. Texturing (النسيج)
   └─ إضافة ألوان وأنماط

5. Rigging (التهيئة)
   └─ إضافة عظام للحركة

6. Animation (الرسوم المتحركة)
   └─ إحياء الحركة

7. Lighting (الإضاءة)
   └─ إضاءة المشهد

8. Rendering (الرسم النهائي)
   └─ إنتاج الصورة النهائية

Blender (مجاني):
├─ نمذجة كاملة
├─ الرسوم المتحركة
└─ Rendering قوي

Autodesk Maya:
├─ احترافي جداً
├─ مستخدم في الأفلام
└─ معايير الصناعة

3ds Max:
├─ نمذجة متقدمة
├─ الألعاب والفيديو
└─ قوي جداً

Cinema 4D:
├─ سهل التعلم
├─ رسومات جميلة
└─ صناعة الإعلان

✅ استخدم مصادر ضوء طبيعية
✅ تجنب الظلال الحادة جداً
✅ جرّب ألوان مختلفة
✅ استخدم العاكسات للضوء الرقيق
❌ تجنب الإضاءة المسطحة
❌ لا تضع المصدر مباشرة أمام الموضوع

✅ اترك مساحة للحركة (Action Space)
✅ استخدم قاعدة الأثلاث
✅ فكر في الخطوط والأشكال
✅ تجنب مركزية الموضوع الدائمة
❌ لا تضع الموضوع الرئيسي في المنتصف دائماً
❌ لا تجعل الخلفية مزعجة

DaVinci Resolve (مجاني):
├─ تحرير احترافي
├─ مكامل عالي الجودة
└─ موجود على كل الأنظمة

Adobe Premiere Pro:
├─ معيار الصناعة
├─ قوي جداً
└─ مشترك

Final Cut Pro:
├─ خاص بـ macOS
├─ سريع جداً
└─ مدفوع مرة واحدة

Adobe Lightroom:
├─ تحرير محترف
├─ تنظيم الصور
└─ مشترك

Capture One:
├─ جودة عالية
├─ تحكم كامل
└─ احترافي

Affinity Photo:
├─ بديل Photoshop
├─ رخيص نسبياً
└─ قوي جداً
