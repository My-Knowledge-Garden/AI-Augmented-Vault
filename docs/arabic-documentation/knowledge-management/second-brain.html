Second Brain

نظام خارجي يحفظ ويعالج المعلومات بدلاً عن عقلك البيولوجي.

الأهداف:

  • تحرير عقلك للتفكير الإبداعي
  • حفظ آمن للمعلومات
  • استرجاع سريع وسهل
  • إنشاء معرفة جديدة
Input (الإدخال):
├─ قراءة الكتب
├─ الاستماع للبودكاست
├─ المقالات والدراسات
└─ التجارب الشخصية

Processing (المعالجة):
├─ التلخيص
├─ الفهرسة
├─ الربط
└─ التحليل

Output (الإخراج):
├─ استرجاع سريع
├─ إنشاء محتوى
├─ اتخاذ قرارات
└─ ابتكار أفكار

Digital Tools:
├─ Obsidian (ملاحظات مترابطة)
├─ Notion (قاعدة بيانات شاملة)
├─ Roam Research (شبكة معرفية)
└─ OneNote (تنظيم بسيط)

Physical Tools:
├─ دفاتر جيدة
├─ أقلام ملونة
└─ نظام ملفات منظم
  1. اختر أداة مناسبة
  2. بدّ نظاماً بسيطاً
  3. ابدأ بالإدخال المنتظم
  4. طوّر النظام تدريجياً
  5. راجع وحسّن
