# صفحات ويب بسيطة

## إنشاء صفحات ويب احترافية وفعالة

---

## 🌐 أساسيات الويب

**التقنيات الثلاث:**
- HTML: الهيكل
- CSS: التنسيق والتصميم
- JavaScript: التفاعل والديناميكية

---

## 📝 هيكل صفحة ويب

```html
<!DOCTYPE html>
<html>
  <head>
    <title>عنوان الصفحة</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <!-- المحتوى هنا -->
    <script src="script.js"></script>
  </body>
</html>
```

---

## 🎨 التصميم الجيد

```
✅ بسيط وواضح
✅ سريع التحميل
✅ يتكيف مع جميع الأجهزة
✅ سهل الاستخدام
✅ جميل وعصري
```

---

## 🚀 خطوات البدء

1. تعلم HTML الأساسي
2. تعلم CSS للتنسيق
3. أضف JavaScript للتفاعل
4. اختبر على متصفحات مختلفة
5. انشرها على الويب
