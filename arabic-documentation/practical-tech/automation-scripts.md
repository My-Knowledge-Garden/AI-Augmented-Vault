# سكريبتات الأتمتة

## أتمتة المهام المتكررة باستخدام البرمجة

---

## ⚙️ ما هي الأتمتة؟

تقليل العمل اليدوي بإنشاء برامج تقوم بالعمل تلقائياً.

**الفوائد:**
- ✅ توفير الوقت
- ✅ تقليل الأخطاء
- ✅ زيادة الإنتاجية
- ✅ عمل متسق

---

## 🛠️ لغات الأتمتة

```
Python:
├─ Automation scripts
├─ سهل التعلم
└─ مرن جداً

Bash/Shell:
├─ Linux automation
├─ قوي جداً
└─ مدمج في النظام

JavaScript:
├─ Web automation
├─ browser automation
└─ تطبيقات ويب
```

---

## 💡 أمثلة أتمتة

- نسخ الملفات تلقائياً
- إرسال رسائل بريد إلكترونية
- تحديث قواعد البيانات
- جدولة المهام
