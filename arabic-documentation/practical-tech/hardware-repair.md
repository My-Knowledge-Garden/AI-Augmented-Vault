# إصلاح الأجهزة الإلكترونية

## تشخيص وإصلاح مشاكل الحواسيب والأجهزة

---

## 🖥️ مشاكل شائعة

**الحاسوب بطيء:**
- ✓ تنظيف الملفات المؤقتة
- ✓ إلغاء البرامج غير المستخدمة
- ✓ تحديث البرامج
- ✓ فحص الفيروسات

**عدم الانطلاق:**
- ✓ فحص الطاقة
- ✓ إعادة التشغيل
- ✓ دخول الـ BIOS
- ✓ استرجاع من نسخة احتياطية

---

## 🔍 خطوات التشخيص

```
1. وصف المشكلة بدقة
2. فحص الاتصالات
3. اختبر الأجزاء واحدة
4. ابحث عن رسائل الخطأ
5. استخدم أدوات تشخيص
```

---

## 🛠️ الأدوات الأساسية

- مفكات مختلفة
- مضاد إلكتروستاتيكي (ESD Strap)
- مصباح يدوي
- أدوات اختبار (Multimeter)
