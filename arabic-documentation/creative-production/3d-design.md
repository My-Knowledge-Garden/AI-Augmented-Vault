# التصميم ثلاثي الأبعاد

## شرح شامل للنمذجة والرسوم المتحركة والتصيير

---

## 1️⃣ مقدمة في التصميم ثلاثي الأبعاد

التصميم ثلاثي الأبعاد (3D Design) هو إنشاء أشياء افتراضية في بيئة حاسوبية ثلاثية الأبعاد.

**التطبيقات:**
- 🎬 **أفلام وحركات**: كل المؤثرات البصرية
- 🎮 **ألعاب**: جميع الأصول والبيئات
- 🏗️ **العمارة**: محاكاة البنايات
- 👔 **الموضة والمنتجات**: عرض المنتجات
- 📺 **الإعلان**: فيديوهات احترافية
- 🏥 **الطب**: محاكاة طبية

---

## 2️⃣ أساسيات البرامج

### 2.1 برامج 3D الشهيرة

```
Blender (مجاني ومفتوح المصدر):
├─ نمذجة وتحريك شاملة
├─ Rendering قوي (Cycles)
├─ مجتمع ضخم
└─ الأفضل للمبتدئين

Autodesk Maya (احترافي/مدفوع):
├─ معيار الصناعة
├─ أفلام والألعاب
└─ قوي جداً

3ds Max (احترافي/مدفوع):
├─ نمذجة متقدمة
├─ الألعاب والفيديو
└─ قوي جداً

Cinema 4D (احترافي/مدفوع):
├─ جميل وسهل
├─ حركة رسومات
└─ الإعلان والعرض
```

### 2.2 واجهة Blender الأساسية

```
3D Viewport (المشهد):
├─ حيث تشتغل على النموذج
├─ تحريك الماوس للتنقل
└─ Mouse wheel للتكبير/التصغير

Outliner (قائمة العناصر):
├─ تسلسل كل الأشياء في المشهد
├─ إظهار/إخفاء العناصر
└─ تنظيم وإدارة

Properties Panel (الخصائص):
├─ إعدادات العناصر المختارة
├─ التحكم بالموضع والحجم
└─ الألوان والمواد

Timeline (الخط الزمني):
├─ للرسوم المتحركة
├─ Keyframes وتسلسل الحركة
└─ تشغيل والقراءة
```

## 3️⃣ عملية النمذجة

### 3.1 الخطوات الأساسية

```
1. Concept Art (الفن الاستكشافي)
   └─ رسومات تخطيطية سريعة

2. Blocking Out (البناء الأساسي)
   └─ أشكال بسيطة (مكعبات، كرات)

3. Refinement (التحسين)
   └─ إضافة حواف وتفاصيل أساسية

4. Detailing (التفاصيل العميقة)
   └─ مسام، خدوش، نعوتات

5. Cleanup (التنظيف)
   └─ إزالة الأخطاء والعيوب

6. Optimization (التحسين)
   └─ تقليل العدد الرؤوس (Geometry)
```

### 3.2 أنماط النمذجة

```
Polygon Modeling (النمذجة الحسابية):
├─ تعديل الرؤوس والحواف والوجوه
├─ الأكثر شيوعاً
└─ سيطرة كاملة

Box Modeling (نمذجة الصناديق):
├─ تبدأ بمكعب أساسي
├─ تعديله تدريجياً
└─ سهل للمبتدئين

Sculpting (النحت):
├─ تنحت الشكل مثل الطين
├─ تفاصيل عميقة
└─ برامج مثل ZBrush

Procedural Modeling (نمذجة إجرائية):
├─ استخدام خوارزميات
├─ تفاصيل تلقائية
└─ فعالة جداً
```

## 4️⃣ المواد والألوان (Texturing)

### 4.1 أنواع الخرائط

```
Diffuse/Albedo Map:
├─ لون السطح الأساسي
├─ بدون إضاءة أو ظلال
└─ الخريطة الأساسية

Normal Map:
├─ تفاصيل السطح الدقيقة
├─ يعطي إحساس بالعمق
└─ بدون تعقيد الجيومتري

Roughness Map:
├─ تحديد نعومة/خشونة السطح
├─ سلس = عاكس
└─ خشن = ممتص للضوء

Metallic Map:
├─ معادن وأسطح معدنية
├─ عاكسة قوية جداً
└─ غير عادي

Ambient Occlusion Map:
├─ ظلال في الشقوق
├─ عمق بصري
└─ توقيع طبيعي
```

### 4.2 أدوات Texturing

```
Substance Painter:
├─ الأفضل للنسيج الديجيتالي
├─ احترافي جداً
└─ مشترك

Blender Shading:
├─ نظام مواد قوي
├─ مجاني
└─ Nodes وإجرائي

Quixel Megascans:
├─ محتوى نسيج احترافي
├─ مكتبة ضخمة
└─ مشترك أو مجاني

Texturing.xyz:
├─ نسيج احترافي
├─ اشتراك معقول
└─ جودة عالية
```

## 5️⃣ الرسوم المتحركة (Animation)

### 5.1 مبادئ الرسوم المتحركة

```
التوقيت والتأخير (Timing & Spacing):
├─ الحركة الطبيعية تحتاج توقيت
└─ المسافات بين الـ Keyframes مهمة

التنبؤ (Anticipation):
├─ قبل الحركة الرئيسية حركة تحضير
├─ مثل الثني قبل القفز
└─ تبدو طبيعية أكثر

الانسحاق والتمدد (Squash & Stretch):
├─ تبدو حية وديناميكية
├─ كرة ترتد: تنسحق وتتمدد
└─ أساسي في الرسوم المتحركة

المتابعة والتراكب (Follow Through):
├─ الأجزاء تستمر بعد توقف الحركة الرئيسية
├─ الشعر يتحرك بعد الرأس
└─ الملابس والقماش

الرجوع (Arcs):
├─ الحركة في منحنيات، لا خطوط مستقيمة
├─ طبيعية أكثر
└─ أساسي للحركة
```

### 5.2 مراحل الرسوم المتحركة

```
1. Rigging (التهيئة):
   └─ إضافة عظام وتحكم للحركة

2. Blocking (الكتل):
   └─ حركات أساسية، بدون تفاصيل

3. Spline (التنعيم):
   └─ تنعيم الحركة بين الـ Keyframes

4. Polish (الصقل):
   └─ تفاصيل وحركات ثانوية

5. Review (المراجعة):
   └─ تصحيح الأخطاء والتحسينات
```

## 6️⃣ الإضاءة والـ Rendering

### 6.1 أنواع الإضاءة

```
Three-Point Lighting:
├─ Key Light (الضوء الرئيسي)
├─ Fill Light (ضوء الملء)
└─ Back Light (الضوء الخلفي)

Image Based Lighting (IBL):
├─ إضاءة من صورة بيئية
├─ واقعية عالية
└─ محاكاة سماء

Area Lights:
├─ مساحة مضيئة
├─ ظلال ناعمة
└─ واقعية

Point Lights:
├─ نقطة مضيئة
├─ ظلال حادة
└─ مثل المصابيح
```

### 6.2 محركات Rendering

```
Cycles (Blender):
├─ مسار تتبع (Path Tracing)
├─ واقعي جداً
└─ يحتاج وقت

Eevee (Blender):
├─ بيانات فعلية (Real-time)
├─ سريع جداً
└─ معقول للجودة

Arnold:
├─ احترافي جداً
├─ الأفلام والإعلان
└─ قوي جداً

V-Ray:
├─ معيار الصناعة
├─ الأفلام والمعمارية
└─ احترافي
```

## 7️⃣ الممارسات الفضلى

### 7.1 نصائح النمذجة

```
✅ ابدأ بـ Reference Images (صور مرجعية)
✅ استخدم Proportions صحيحة
✅ ركز على Silhouette (الشكل الخارجي)
✅ تجنب الهندسة غير الضرورية
✅ حافظ على Topology نظيفة

❌ لا تأضف تفاصيل بدون داعي
❌ لا تحتفظ بـ Geometry قديم
❌ لا تتجاهل النسب
❌ لا تستعجل الخطوات
```

### 7.2 نصائح Texturing

```
✅ استخدم PBR Workflow
✅ احفظ Diffuse منفصل
✅ تحقق من الإضاءة المختلفة
✅ استخدم مكتبات جودة عالية
✅ تجنب الجودة الزائدة

❌ لا تخلط الإضاءة مع الخريطة
❌ لا تستخدم جودة عالية جداً
❌ لا تتجاهل Normal Maps
❌ لا تنسى Roughness
```
