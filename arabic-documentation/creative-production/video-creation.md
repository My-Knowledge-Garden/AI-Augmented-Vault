# إنشاء الفيديو

## دليل شامل لإنشاء وتحرير الفيديو الاحترافي

---

## 1️⃣ أساسيات إنشاء الفيديو

### 1.1 من أين تبدأ؟

**المعدات الأساسية:**
```
الحد الأدنى:
├─ هاتف ذكي (كاميرا جيدة)
├─ ميكروفون خارجي (اختياري)
└─ حامل ثابت (Tripod)

الاحترافي:
├─ كاميرا DSLR أو Mirrorless
├─ عدسات متعددة
├─ ميكروفون احترافي
├─ إضاءة احترافية
└─ جيمبال (Gimbal) أو Slider
```

### 1.2 أنواع الفيديو

```
Vlogs (فيديوهات يومية):
├─ شخصية وعفوية
├─ 5-15 دقيقة
└─ هاتف ذكي كافي

Tutorials (دروس):
├─ تعليمية منظمة
├─ 5-30 دقيقة
└─ كاميرا + محرر

Cinematic Videos (فيديوهات سينمائية):
├─ احترافية ودرامية
├─ 2-5 دقائق
└─ معدات احترافية

Live Streaming (البث المباشر):
├─ فوري وتفاعلي
├─ طول متغير
└─ كاميرا + انترنت جيد
```

## 2️⃣ مراحل الإنتاج

### المرحلة 1: الإعداد (Pre-Production)

```
1. الكتابة السيناريو:
   └─ اكتب كل ما ستقوله
   
2. اللوحات المرسومة (Storyboard):
   └─ رسم كل مشهد قبل التصوير
   
3. تخطيط الموقع:
   └─ تحضير المكان والإضاءة والمعدات
   
4. تجهيز الممثلين/المضيفين:
   └─ ملابس، ماكياج، تدريب
   
5. قائمة التحقق:
   └─ تأكد من كل شيء قبل البدء
```

### المرحلة 2: التصوير (Production)

```
نصائح التصوير:
✅ اختبر الإضاءة أولاً
✅ اختبر الصوت قبل التسجيل
✅ صوّر أكثر من اللازم (Footage إضافي)
✅ استخدم حامل ثابت
✅ بطء الحركة = صور أناعم
✅ اترك فراغات بين المشاهد

تجنب:
❌ الإضاءة السيئة
❌ الصوت المزعج
❌ الحركة المرتجة
❌ الملابس الساطعة جداً
```

### المرحلة 3: التحرير (Post-Production)

```
1. الفرز والتنظيم:
   └─ نظّم كل الملفات
   
2. التحرير الخام (Rough Cut):
   └─ ضع المشاهد بالترتيب
   
3. التحرير الدقيق (Fine Cut):
   └─ اضبط التوقيت
   
4. إضافة المؤثرات والموسيقى:
   └─ ألوان، انتقالات، صوت
   
5. الترجمة والنصوص:
   └─ أضف ترجمات وعناوين
   
6. الإتقان النهائي:
   └─ تحقق من كل شيء
   
7. التصدير:
   └─ احفظ بصيغ مختلفة
```

## 3️⃣ تقنيات التصوير

### 3.1 الحركات الكاميرا

```
Static Shot (ثابت):
├─ الكاميرا لا تتحرك
├─ بسيط وفعال
└─ مناسب للمقابلات

Pan (مسح أفقي):
├─ الكاميرا تتحرك جانباً
├─ مناسب لإظهار منظر عريض
└─ ببطء وسلاسة

Tilt (مسح عمودي):
├─ الكاميرا تتحرك لأعلى/أسفل
├─ تظهر حجم شيء
└─ ببطء وتدريجي

Zoom (تقريب):
├─ تقريب أو إبعاد
├─ يجب أن يكون بطيء
└─ تجنب القفز

Dolly (تقريب الكاميرا):
├─ تقريب الكاميرا من الموضوع
├─ يخلق عمق ودراما
└─ سلس وتدريجي
```

### 3.2 الزوايا والإطارات

```
Wide Shot (لقطة عريضة):
├─ إظهار السياق
├─ مشهد عام
└─ لقطة إنشائية

Medium Shot (لقطة متوسطة):
├─ من الخصر لأعلى
├─ معيارية مشتركة
└─ ودية وقريبة

Close-up (لقطة قريبة):
├─ من الرقبة لأعلى
├─ تظهر الحواس والمشاعر
└─ درامية وقوية

Over-the-Shoulder (فوق الكتف):
├─ نظر من خلف شخص
├─ تفاعل بين شخصين
└─ طبيعية وواقعية
```

---

## 4️⃣ الصوت والموسيقى

### 4.1 جودة الصوت

```
معايير جيدة:
✅ خالي من الضوضاء الخارجية
✅ صوت واضح
✅ مستوى صوت متساوي
✅ بلا تشويه

طرق التحسين:
├─ استخدم ميكروفون خارجي
├─ سجل في مكان هادئ
├─ تجنب الأماكن ذات الصدى
└─ راقب مستويات الصوت أثناء التسجيل
```

### 4.2 الموسيقى والمؤثرات

```
موسيقى خلفية:
├─ اختر موسيقى تناسب المشهد
├─ استخدم موسيقى بدون حقوق (Royalty-free)
├─ اجعلها هادئة (Background)
└─ لا تطغ على الكلام

مؤثرات صوتية:
├─ تصفيق، خطوات، أصوات بيئية
├─ تضيف واقعية
├─ استخدم بحذر
└─ لا تبالغ

مواقع موسيقى مجانية:
├─ YouTube Audio Library
├─ Epidemic Sound
├─ Artlist
└─ Free Music Archive
```

---

## 5️⃣ التحرير المتقدم

### 5.1 مؤثرات بصرية

```
الألوان (Color Grading):
├─ دافئة: ذهبية وحنونة
├─ باردة: زرقاء وهادئة
├─ سينمائية: تباين عالي
└─ مرشحات: بسيطة وجميلة

الانتقالات:
├─ Cut: قطع مباشر
├─ Fade: تلاشي
├─ Dissolve: ذوبان
├─ Wipe: مسح بصري
└─ Zoom: تقريب

الرسوم المتحركة:
├─ عناوين متحركة
├─ نصوص بأثر متحرك
├─ أسهم وأشكال
└─ انتقالات سلسة
```

### 5.2 الترجمة والشرح

```
متى تستخدم:
✅ عندما تكون الجودة الصوتية سيئة
✅ لوسائل التواصل (بدون صوت)
✅ للأغاني الأجنبية
✅ لمساعدة الصم والبكم

كيفية الإضافة:
├─ استخدم برامج مثل Adobe Premiere
├─ أو تطبيقات مثل Kapwing
├─ ركز على التوقيت الصحيح
└─ احرص على وضوح الخط
```

## 6️⃣ توزيعات الفيديو المختلفة

### 6.1 التقارير والأحجام

```
YouTube:
├─ Full HD (1080p)
├─ 16:9 (عريض)
└─ 60 ثانية على الأقل

Instagram:
├─ 1080 x 1080 (مربع)
├─ 1080 x 1920 (عمودي)
└─ 3-60 ثانية

TikTok:
├─ 1080 x 1920 (عمودي)
├─ 9:16 (عمودي)
└─ 15 ثانية - دقائق

Facebook:
├─ 16:9 أو 1:1
├─ Full HD
└─ 30+ ثانية مثالي

LinkedIn:
├─ 16:9
├─ HD على الأقل
└─ احترافي وتعليمي
```

### 6.2 إعدادات التصدير

```
للويب:
├─ MP4
├─ 1080p
├─ H.264 codec
└─ 8 Mbps bitrate

للقنوات:
├─ MP4 / MOV
├─ 2K أو 4K
├─ ProRes codec (احترافي)
└─ Uncompressed (عالي الجودة)
```

## 7️⃣ نصائح واحترافية

### 7.1 الإضاءة

```
✅ استخدم شمس غروب طبيعية
✅ ثلاث نقاط إضاءة معيارية
✅ العاكسات توزع الضوء
✅ تجنب الظلال القاسية

❌ لا تصوّر أسفل الشمس مباشرة
❌ لا تستخدم الإضاءة الخلفية فقط
❌ تجنب مسطح الإضاءة
```

### 7.2 الصوت

```
✅ اختبر الصوت قبل بدء التسجيل
✅ استخدم ميكروفون خارجي دائماً
✅ راقب المستويات باستمرار
✅ صوّر صوت اختبار أولاً

❌ لا تصوّر بدون اختبار
❌ لا تتجاهل الأصوات الخارجية
❌ لا تبالغ في مستوى الصوت
```
