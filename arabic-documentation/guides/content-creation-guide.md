## ✍️ خطوات إنشاء محتوى

```
1. اختر الفئة الصحيحة
   ├─ 1-creative-production
   ├─ 2-radical-discourse
   ├─ 3-practical-tech
   └─ 4-knowledge-management

2. أنشئ ملف Markdown جديد
   └─ استخدم params-case للاسم

3. أضف Front Matter
   ├─ title
   ├─ description
   ├─ category
   ├─ tags
   ├─ public: true
   └─ author

4. اكتب المحتوى
   ├─ هيكل واضح
   ├─ أمثلة عملية
   └─ روابط مفيدة

5. حدّث manifest
   └─ node generate-manifest.js
```
